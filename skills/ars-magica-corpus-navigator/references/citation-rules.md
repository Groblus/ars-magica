# Citation Rules

Use citations for all source-backed claims.

Required citation format:

```text
reviewed/<book>.md:line_start-line_end
```

Rules:

- Prefer Definitive Edition core citations for rules.
- Use 5e citations for supplement-specific depth.
- Never cite 3e or 4e books from this skill.
- If search results conflict, read the source lines before answering.
- If a line span is generated from the index, verify it if the answer depends on exact wording.
