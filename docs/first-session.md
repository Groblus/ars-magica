# Ars Magica First-Session Packet

Use Definitive Edition core as rules authority. Use 5th Edition sourcebooks as compatible support. Keep 3e/4e books out of first-session rules lookup.

## Before Session

- Ask each player for one concept sentence: house, magical obsession, social trouble, and one mundane tie.
- Decide if characters are full magi, apprentices, companions, or a mixed covenant cast.
- Pick saga tone: wondrous scholarship, political covenant drama, faerie weirdness, holy frontier, infernal threat, or rural survival.
- Pick one tribunal or keep geography loose until covenant creation.
- Read only core basics first: character creation, Hermetic magic, covenants, obstacles, stories, sagas.

## Table Decisions

| Decision | Default For First Play |
|---|---|
| Rules depth | Use core mechanics, defer edge cases. |
| Character scope | Start with magi plus simple companion/grog sketches. |
| Covenant | Build a light covenant pitch, not full accounting. |
| Magic | Encourage formulaic spells plus one signature spontaneous trick. |
| Advancement | Explain seasons, but do not run full lab/season cycle in session one. |

## Player Primer

Ars Magica is troupe fantasy about a covenant of wizards and their allies in Mythic Europe. Magi are powerful but socially constrained by the Order of Hermes, local politics, the Church, supernatural realms, and their own long-term research ambitions.

Core loop: stories interrupt long-term plans. Between stories, characters study, invent spells, train, age, improve labs, and develop the covenant.

## Opening Hooks

1. A redcap arrives wounded with a sealed message naming the covenant.
2. A local priest asks for help with miracles that look wrong.
3. A noble patron offers land if the magi solve a border haunting.
4. A child displays impossible magic near a contested vis source.
5. A faerie demands that the covenant finish an old story.

## Rules Cheat Sheet

- Roll stress dice when danger, pressure, or botch risk matters.
- Most actions are characteristic + ability + die against an ease factor.
- Hermetic magic uses Technique + Form + stamina/intelligence/context modifiers.
- Formulaic spells are reliable known spells; spontaneous spells are flexible but weaker.
- Magic resistance and penetration matter when spells affect protected beings.
- Seasons are strategic turns: study, lab work, adventure consequences, covenant projects.
- Covenant resources create both power and story obligations.
