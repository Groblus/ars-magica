What is in the lab

For reading

GitHub Pages hosts a browsable shelf for the Definitive Edition core and 5th Edition sourcebooks, with line-linked tables of contents and searchable core rules tables.

For agents

The repo includes an installable corpus navigator skill, generated Markdown references, a SQLite FTS database, OpenAI large embeddings, and sqlite-vector support.

For play prep

The current baseline prioritizes first-time play: core rules first, 5e support books second, and 3e/4e material kept out of active rules lookup.
