First covenant night

Start small. Let Mythic Europe get strange fast.

A practical packet for a first Ars Magica session: table decisions, concept prompts, covenant seeds, opening hooks, agenda, rules cheat sheet, and source pointers.

Open Reference Library Markdown Version
DE Definitive Edition core rules first. 5e books as support. Legacy editions out of first-session mechanics.

Before Session

  • Ask each player for one concept sentence: house, magical obsession, social trouble, and one mundane tie.
  • Pick saga tone: wondrous scholarship, political covenant drama, faerie weirdness, holy frontier, infernal threat, or rural survival.
  • Read only core basics first: characters, Hermetic magic, covenants, obstacles, stories, sagas.

Default Scope

  • Start with magi and simple companion/grog sketches.
  • Build a light covenant pitch, not full accounting.
  • Explain seasons, but defer full lab/season cycles until after the opening story.

Player Primer

Ars Magica is troupe fantasy about a covenant of wizards and allies in Mythic Europe. Stories interrupt long-term plans; seasons turn consequences into study, lab work, politics, and covenant growth.

Opening Hooks

  1. A redcap arrives wounded with a sealed message naming the covenant.
  2. A local priest asks for help with miracles that look wrong.
  3. A noble patron offers land if the magi solve a border haunting.
  4. A child displays impossible magic near a contested vis source.
  5. A faerie demands that the covenant finish an old story.

Rules Cheat Sheet

  • Most actions are characteristic + ability + die against an ease factor.
  • Use stress dice when danger, pressure, or botch risk matters.
  • Hermetic magic uses Technique + Form plus context modifiers.
  • Formulaic spells are reliable; spontaneous spells are flexible but weaker.
  • Magic resistance and penetration matter against protected beings.
  • Seasons are strategic turns for study, lab work, and covenant projects.
