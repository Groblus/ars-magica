# Ars Magica MCP Server

Local FastMCP server for the generated Ars Magica SQLite reference database.

```bash
python3 mcp/ars_magica_server.py
```

Tools expose FTS lookup, section retrieval, book TOCs, spells, virtues, flaws, abilities, and covenant boons/hooks. They use only the local database and do not call OpenAI.
